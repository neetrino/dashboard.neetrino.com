# Dashboard Control System

Эта папка содержит все файлы, связанные с централизованным управлением плагином Neetrino через Dashboard Control Panel.

## Файлы в папке:

### 1. `Dashboard_Connect.php`
- **Назначение**: Основной класс для подключения к Dashboard
- **Функции**:
  - Автоматическая регистрация плагина в Dashboard
  - Отправка периодических ping-запросов (каждые 10 секунд)
  - Обработка команд от Dashboard (включение/отключение модулей, удаление плагина)
  - Управление статусом подключения

### 2. `Registration.php`
- **Назначение**: Класс для регистрации плагина с Dashboard
- **Функции**:
  - Единственный метод регистрации для всех случаев
  - Генерация временных ключей безопасности
  - Отправка данных о сайте на Dashboard
  - Очистка старых данных регистрации

### 3. `REST_API.php`
- **Назначение**: REST API для мгновенного выполнения команд (push-архитектура)
- **Функции**:
  - Регистрация REST маршрутов WordPress
  - Выполнение команд от Dashboard в реальном времени
  - Система безопасности (проверка API ключей и IP)
  - Управление модулями, удаление плагина, получение статуса

## Система безопасности:

1. **API ключи** - уникальные ключи для каждого сайта
2. **IP проверки** - доступ только с IP Dashboard'а
3. **Временные ключи** - для первичной регистрации
4. **Логирование** - все действия записываются в лог

## Инициализация:

Все классы инициализируются в основном файле плагина `neetrino.php`:

```php
// Dashboard Control System - файлы для управления с dashboard
require_once plugin_dir_path(__FILE__) . 'includes/dashboard-control/Registration.php';
require_once plugin_dir_path(__FILE__) . 'includes/dashboard-control/Dashboard_Connect.php';
require_once plugin_dir_path(__FILE__) . 'includes/dashboard-control/REST_API.php';

// Инициализация Dashboard Connect
Neetrino_Dashboard_Connect::init();
// REST API инициализируется автоматически в классе
```

## Взаимодействие с Dashboard:

1. **Регистрация** - происходит автоматически при активации плагина
2. **Мониторинг** - постоянный обмен данными каждые 10 секунд
3. **Управление** - мгновенное выполнение команд через REST API
4. **Безопасность** - многоуровневая система защиты
